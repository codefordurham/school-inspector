About Us

Please take our user survey to help design Phase 2 of this project!

This website is a project of Code for Durham, a local volunteer brigade of the national non-profit Code for America. We are a collection of public school parents and concerned citizens with who want help parents understand the public school system in Durham by supplementing existing DPS web resources.

Our goals in this effort are two-fold:
  • Help parents understand the public schools available to their family and make informed schooling decisions based on accurate and complete information
  • Provide a common platform for all public schools to share information about their offerings and accomplishments with parents.
Through this project, we strive to uphold our core values:
  • Collaboration
  • Transparency
  • User-friendliness
The project is divided into 2 phases:
  • Phase 1: Clarify school zoning questions with an intuitive mapping tool.
  • Phase 2: Inform school decisions through comprehensive profiles that include information about each schools' curriculum and philosophy, lottery process, academic performance, teacher retention and much more.

We are currently collecting feedback for phase one and user input for phase two. Please note that we have not yet formally launched the site and are still testing for accuracy and completeness.

In many respects, this website could be considered an economic development tool, helping make it easier for parents to make informed choices about moving to Durham versus other surrounding jurisdictions. Code for Durham projects are all open source and we encourage and welcome contributions in the form of user testing, school system partnership, and web development support.
